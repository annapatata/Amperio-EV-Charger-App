#An introductory example for the project

To check it you need to have installed node.js
After that go to 'example/' and run npm install and to the same in the 'client/'
In two separate terminal do "node server.js" in the 'example/' and "npm start" in the 'client/'.

You also need to have out database and run in for the server to retrieve the data.
After that, create a .env file to apadt the code in server.js accordingly.
